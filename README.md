# RooMN
Roomn is a roommate-finding and subleasing platform designed to help students browse, save, and post housing listings in a simple and organized way. The project focuses on building a clean frontend experience while exploring real-world deployment, infrastructure, and CI/CD workflows.
## Live Demo
https://brave-ground-0fb31fe1e.6.azurestaticapps.net/

## Motivation
Finding roommates and short-term housing is often fragmented across group chats and spreadsheets. RooMN was created as an experiment in building a centralized, intuitive interface for browsing and managing listings, while also serving as a learning project in frontend architecture and cloud deployment.

## Features
- Browse housing listings in a clean grid-based layout
- Filter listings by preferences and criteria
- Save listings for later review
- Post new listings through a structured form
- Profile and saved listings views
- Responsive, mobile-friendly UI
- **Note:** The current deployment is a static frontend preview. Backend and authentication features are planned.
    
## Tech Stack
- Frontend: Next.js (React, TypeScript)
- Styling: CSS Modules
- Deployment: Azure Static Web Apps
- CI/CD: GitHub Actions
- Version Control: Git & GitHub
## Project Status
RooMN is currently in an active development/preview state.
- Planned future work includes:
- Backend API integration
- User authentication and profiles
- persistent data storage
- Listing moderation and validation
- Expanded filtering and search capabilities
